== Portfolio App

This is a portfolio app built in Ruby on Rails that uses BDD and covers several topics including:

* Testing with MiniTest

* Authentication with Devise

* Authorization with Pundit

* Continuous Integration

* Blog functionality (articles, comments, etc.)

* And more!
